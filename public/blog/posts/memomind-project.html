MemoMind — 一款基于 .NET 的备忘录应用

2026-04-22  ·  ⚙️ 开发  ·  约 5 分钟

软件工程课程的期末项目，我做了这款名为 MemoMind 的备忘录应用。

项目简介

MemoMind 是一款使用 C# 和 .NET 开发的桌面备忘录应用，支持：

  • ✅ 创建、编辑和删除待办事项
  • 🏷️ 标签分类管理
  • ⏰ 提醒功能
  • 🔍 搜索和筛选

技术栈

项目使用 C# / WPF (Windows Presentation Foundation) 构建前端界面，后端使用 SQLite 作为本地数据库。

收获与心得

这次项目让我对 C# 和 .NET 有了更深入的理解。从需求分析、架构设计到编码实现和测试，完整地经历了一次软件开发的流程。

项目代码已经开源在 GitHub 上，欢迎查看和交流！
