用 GitHub Pages 搭建个人网站

2025-10-15  ·  💻 技术  ·  约 5 分钟

GitHub Pages 是 GitHub 提供的免费静态网站托管服务。用它来搭建个人网站再合适不过了——免费、快速、无需管理服务器。

1. 创建仓库

在你的 GitHub 上创建一个名为 <username>.github.io 的仓库（Public 仓库），这就是你的网站仓库了。

2. 克隆到本地

git clone https://github.com/<username>/<username>.github.io.git
cd <username>.github.io

3. 添加你的网站文件

在仓库根目录下创建 index.html，这就是你的网站首页。你还可以创建其他页面和目录。

4. 推送上线

git add .
git commit -m "初始化个人网站"
git push origin main

推送完成后，等待 1-2 分钟，你的网站就会在 https://<username>.github.io 上线了！

5. 自定义域名（可选）

在仓库设置中启用 GitHub Pages，并添加你的自定义域名。然后在域名 DNS 中添加一条 CNAME 记录指向 <username>.github.io。

这样就完成了！你就可以拥有一个属于自己的网站啦 🎉
